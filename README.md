# Crockpot
## Overview
This is an Android application which provides a comprehensive list of every recipe in the popular game Don't Starve. Users can save their favorite recipes, upload their own recipe ideas along with a photo, and view community-submitted recipes.

## Getting Started
### Prerequisites
Before you begin, ensure you have the following installed:

1. [Android Studio Iguana](https://developer.android.com/studio/releases/past-releases/as-iguana-release-notes)
2. You have set up a [Firebase project](https://firebase.google.com/) and have the google-services.json file.

### Installation
1. Place the google-services.json file in the app/ directory of the project.
2. Sync the project with Gradle files
